Anthropic: От чат-бота к рабочей среде

Claude 3.5 Sonnet получил функцию "Artifacts"

26 июня 2025 Интерфейс Продуктивность

Это ключевое обновление для Anthropic. Теперь, когда Claude генерирует код, текст или дизайн, результат появляется не просто в чате, а в специальном окне "Artifacts" рядом. В этом окне можно редактировать результат, и Claude тут же увидит изменения и адаптирует свои дальнейшие ответы.

Что это значит: Anthropic превращает свой продукт из простого чат-бота в интерактивную рабочую среду (IDE). Это прямой шаг к созданию полноценных AI-агентов, которые не просто отвечают на вопросы, а совместно с пользователем работают над проектом в реальном времени. Это меняет парадигму "запрос-ответ" на "совместную работу".

OpenAI: Глубокая интеграция и API

Подтверждена глубокая интеграция GPT-4o в iOS 19

24 июня 2025 Партнерство Мобильный AI

Apple и OpenAI официально подтвердили детали партнерства. GPT-4o будет интегрирован в Siri и системные инструменты (например, "Почта" и "Заметки"). Простые запросы Siri будет обрабатывать на устройстве, а для сложных (например, "напиши стихотворение в стиле Шекспира о моей собаке") она будет обращаться к API OpenAI, предварительно спросив разрешение у пользователя. Конфиденциальность — ключевой аспект: IP-адреса не будут логироваться, а запросы не будут использоваться для обучения моделей OpenAI.

Новые возможности fine-tuning API для GPT-4

28 июня 2025 Для разработчиков API

OpenAI выкатила обновление для API дообучения (fine-tuning). Теперь разработчики могут использовать более гранулярные настройки, включая "слоистую" настройку, когда модель дообучается не целиком, а только на определенных "слоях", что значительно дешевле и быстрее. Также добавлена интеграция с "весами", позволяя смешивать оригинальную модель с дообученной в нужной пропорции.

Что это значит: OpenAI делает ставку на кастомизацию. Вместо одной модели для всех, они дают бизнесу инструменты для создания узкоспециализированных, "своих" версий GPT-4, что открывает дорогу для более сложных и точных корпоративных AI-решений.

Google Gemini: Память и эффективность

Функция "Memory" стала доступна всем пользователям Gemini Advanced

27 июня 2025 Функционал UX

Google наконец-то развернула функцию "Memory" (Память) для всех подписчиков Gemini Advanced. Теперь пользователи могут явно попросить Gemini запомнить определенные факты (например, "Моего начальника зовут Алексей, он любит короткие и четкие отчеты" или "Я пишу код на Python и использую FastAPI"). Модель будет использовать эту информацию в последующих диалогах для предоставления более релевантных и персонализированных ответов.

Что это значит: Это попытка решить проблему "амнезии" чат-ботов и сделать общение с AI более естественным и продолжительным. Вместо того чтобы повторять контекст в каждом новом чате, пользователь формирует долгосрочную базу знаний для своей личной AI-модели.

Поисковые Агенты: От ответа к продукту

Perplexity запускает "Pages" для генерации статей

25 июня 2025 Контент AI-агент

Поисковик Perplexity сделал большой шаг вперед, запустив функцию "Pages". Теперь по запросу пользователя AI не просто дает ответ со ссылками, а генерирует полноценную, структурированную веб-страницу (статью, отчет, гайд) с разделами, изображениями и цитатами. Этими страницами можно делиться по прямой ссылке. Пользователь может выбирать аудиторию (например, "для новичков" или "для экспертов") и структуру статьи.

Что это значит: Perplexity становится не просто "отвечателем на вопросы", а инструментом для создания контента. Это прямой вызов блог-платформам и CMS, позволяющий любому человеку быстро создавать качественные и хорошо оформленные материалы на любую тему.

Инструменты для разработчиков: AI в коде и автоматизации

Cursor IDE представил функцию "Auto-Debug"

29 июня 2025 IDE Разработка

AI-ориентированная среда разработки Cursor анонсировала экспериментальную функцию "Auto-Debug". Когда код падает с ошибкой, разработчик может нажать одну кнопку. AI анализирует stack trace, изучает связанный код в проекте, находит вероятную причину и предлагает готовый патч, который можно применить в один клик. Это выводит AI-ассистентов за рамки простого автодополнения кода.

n8n выпустил набор узлов "Agent-in-a-Box"

26 июня 2025 Автоматизация Low-code

Платформа для автоматизации n8n (аналог Zapier для разработчиков) выпустила новый пакет узлов "Agent-in-a-Box". Он позволяет без глубокого программирования создавать сложных, многошаговых AI-агентов прямо в визуальном редакторе. Можно задать агенту цель, предоставить инструменты (например, "доступ к Google Search" или "возможность отправлять email") и задать логику поведения с памятью.

Что это значит: Создание AI-агентов становится доступным не только для Python-разработчиков, но и для широкого круга технических специалистов через low-code платформы. Это демократизирует разработку сложных автоматизированных систем.
